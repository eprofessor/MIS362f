Data



800 Points
Learning Objectives

These exercises introduce concepts used to define how organizations use data in information systems.

After completing this module you will be able to:

  1. Identify and briefly describe the members of the hierarchy of data
  2.  Identify the advantages of the database approach to data management
  3.  Identify the key factors that must be considered when designing a database
  4.  Identify the various types of data models and explain how they are useful in planning a database
  5.  Describe the relational database model and its fundamental characteristics
  6.  Define the role of the database schema, data definition language, and data manipulation language
  7.  Discuss the role of a database administrator and data administrator
  8.  Identify the common functions performed by all database management systems
  9.  Define the term big data and identify its basic characteristics
  10.  Explain why big data represents both a challenge and an opportunity
  11.  Define the term data management and state its overall goal
  12.  Define the terms data warehouse, data mart, and data lakes and explain how they are different
  13.  Outline the extract, transform, load process
  14.  Explain how a NoSQL database is different from an SQL database
  15.  Discuss the whole Hadoop computing environment and its various components
  16.  Define the term in-memory database and explain its advantages in processing big data
