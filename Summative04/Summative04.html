Software



700 Points
Learning Objectives

These exercises explore how software is a key component use by organization information systems.

After completing this module you will be able to:

  1.  Identify and briefly describe the functions of two basic kinds of software.
  2.  Define the term 'sphere of influence', and describe how it can be used to classify software
  3.  Define the basic functions performed by the operating system
  4.  Identify current operating systems that are used for personal, workgroup, and enterprise computing
  5.  Discuss the role of the operating system in embedded systems
  6.  Discuss how application software can support personal, workgroup, and enterprise business objectives
  7.  Identify three basic approaches to developing application software and discuss the pros and cons of each
  8.  Identify programming languages commonly in use today
  9.  Identify several key software issues and trends that have an impact on organizations and individuals.

Exercise 1-Systems Software

After reading chapter 3, you will realize that operating systems, utility and middleware software is essential  to enable the organization hardware to perform required tasks.

Organizations need to ensure that hardware and software is performing optimally.
Tools such as the Microsoft Sysinternals Suite discussed in Chapter 4, section 2 are useful.
For Windows, click Start and type in "Task Manager". On a Mac use the 'Activity Monitor'
Study the various tabs and investigate some of the processes that are running on the computer.

Note: do not make any changes or stop any services or processes-doing so could have unexpected results!

optional-If you want to be real geeky, investigate using Bginfo64.exe to display system data as the computer desktop background!
