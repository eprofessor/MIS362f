Learning Objectives

These exercises introduce concepts used to define how organizations procure hardware for information systems.

After completing this module you will be able to:

  1.  Describe the functions of the primary components of the computer (CLO 3)
  2.  Explain recent innovations in computer processor chips, memory devices, and input/output devices (CLO 3, and 12)
  3.  Discuss the characteristics and usage of single-user and multi-user computer systems (CLO 3)
  4.  Identify some challenges that must be considered in implementing a data center (CLO 3)
  5.  Identify the primary goals of green computing (CLO 12)
Assignment Overview

The goal of this assignment is to familiarize you with how organizations use the latest available hardware, and energy considerations, in order to efficiently and effectively get things done.

Exercise 1-Configuring a Computer

After reading chapter 2, you will note that the components of a computer are similar, but the number of  possible combinations, from small systems to  massive data center, are almost limitless.

Organizations need to balance purchasing newer, more energy efficient computer hardware with purchase price (and other costs to be discussed in later chapters, such as software and training).
Visit the PC Part Picker website, and use the System Builder to configure a desktop computer system for an organization for which you would like to work.
As you choose the components, consider the tradeoffs in computing power, cost and performance that you are making.
  Choose from innovation, reengineering, continuous improvement, outsourcing, offshoring and downsizing-and use the NYT, WSJ or a magazine available from Flipster (remember Formative00?) to read and summarize an article discussing how an organization used one of these to accomplish change.
