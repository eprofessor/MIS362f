Learning Objectives

After completing this assignment you will be familiar with:

Assignment Overview

Note: Save this assignment to your website.

The purpose of this assignment is to learn how decisions can be made by summarizing data with pivot tables.

  • Summarizing data in a spreadsheet
  • Summarizing massive amounts of spreadsheet data

This exercise will investigate pivot table basics.
If your boss asks you to quickly find out which salesperson sold the most cookies in a certain region, could you do it?
